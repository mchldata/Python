
## Overview

These excercies involved writing Python scripts that interact with users, process inputs, and perform calculations. By combining different programming elements, the project demonstrates how basic code can be used to build functional and user-friendly programs.

## Skills Learned

Throughout this project, the following Python skills were developed:

- Using **variables** to store and manage data  
- Using **print()** to display output to users  
- Using **input()** to capture user input  
- Applying **type casting** to convert data types (e.g., string to integer)  
- Writing **if statements** to control program logic and decision-making  
- Using **for loops** to repeat actions a set number of times  
- Using **while loops** to create continuous or conditional loops  
- Performing **basic calculations** within programs  
- Building simple interactive programs that respond to user input  

## Key Features

- Interactive scripts that prompt users for input and provide responses  
- Logical decision-making based on conditions (if/else statements)  
- Repetitive processes using loops to improve efficiency  
- Simple calculations such as totals, averages, or comparisons  
- Clear and readable output using print statements  

## Outcomes

Through this project, basic programming concepts were combined to create structured and functional Python programs. It demonstrates an understanding of how to:

- Build logic into code to solve problems  
- Process and validate user input  
- Automate repetitive tasks using loops  
- Perform calculations and present results clearly
